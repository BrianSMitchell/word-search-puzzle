# Daily Insights Reports

This directory contains automatically generated daily insights reports for the Word Search Puzzle website.

Each report includes:
- **Metrics**: Daily traffic data from Vercel Analytics
- **Recommendation**: An AI-generated action item based on the data

## Usage with AI Coding Assistants

You can reference these reports when working with AI coding tools:

> "Look at the last 7 daily reports and implement the most impactful recommendation"

> "What patterns do you see in the recommendations from this month?"

## Report Schedule

Reports are generated nightly at midnight EST via GitHub Actions.
